# Ahmet Balcı Apps GitHub Pages

Bu klasör GitHub Pages reposunun köküne yüklenecek statik site dosyalarını içerir.

## Yayın URL'leri

- `https://apps.ahmetbalci.com/`
- `https://apps.ahmetbalci.com/sudoku/privacy-policy/`
- `https://apps.ahmetbalci.com/sudoku/terms/`
- `https://apps.ahmetbalci.com/sudoku/support/`
- `https://apps.ahmetbalci.com/sudoku/data-deletion/`

## GitHub Pages Ayarları

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`
- Custom domain: `apps.ahmetbalci.com`
- Enforce HTTPS: açık

## Destek E-postası

Destek e-postası hazır olunca HTML dosyalarındaki "destek e-posta adresi hazır olduğunda" ifadeleri gerçek e-posta adresiyle değiştirilmeli.
